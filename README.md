# 2048-Snake
Snake for [Creoqode](http://www.creoqode.com) 2048 game console

## How to?
Reqiures standard 2048 game console libraries:
 * https://github.com/adafruit/RGB-matrix-Panel.git
 * https://github.com/adafruit/Adafruit-GFX-Library.git
